10 - Review

01) The property of an object to take on many different forms is called ______________.
    polymorphism

02) Polymorphism can be attained through ________ or ________.
    Inheritance or Interfaces

03) Assuming that the Primate class is a parent class of the Lemur class. The following code is an example of __________________ casting.
    Primate primate = lemur;
        Lemur lemur3 = (Lemur)primate;
    object

04) The specific implementation for a method of a child class is called method _______________.
    overriding

05) If a class have multiple methods by same name but different parameters, it is known as method ______________.
    overloading

06) _____________ polymorphism is a process in which a call to an overridden method is resolved at runtime rather than at compile-time.
    Runtime

07) _________________ polymorphism is the same as method overloading.
    compile-time

08) For a binary search of an array to work correctly, the array must already be __________________.
    sorted

09) A linear search is much more efficient than a binary search.
    False

10) Polymorphism is achieved by
    Overriding
