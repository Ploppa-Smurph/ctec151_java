09 - Review

01) _____________ is the process in which a new class is derived from an existing class.
    Inheritance
    
02) Software ______________is the concept of using existing software components to create new ones.
    reuse

03) The original class that is used to derive a new class is called the ________________ class.
    parent

04) The derived class is called the ____________ class or subclass.
    child

05) True or False. Any public method or variable in a parent class can be explicitly referenced by name in the child class and through objects of the child class.
    True

06) True or False: Private methods and variables of the parent class cannot be referenced by name in the child class or through an object of the child class.
    True

07) (True or False) A variable or method declared with protected visibility may be accessed by any class in the same package in addition to being accessible by any derived class.
    True

08) The ______________ keyword is used in a class to refer to its parent class.
    super
    
09) (True or False) Java supports multiple inheritance.
    False

10) A(n) _________________ class can have both abstract and non-abstract (implemented) methods.
    abstract
