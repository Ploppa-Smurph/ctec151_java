CTEC 151 Advanced Java wk08: Arrays


08.01 - Arrays

    - index begins at 0
    - create new: int[] intArray = new int[10]; will create an integer array containing 10 spaces
        - the default value for an int / double array will be 0 for each 'array element' (an array element are the items in the array).


08.02 - Arrays of Objects

    - the array holds a reference to the object
    - create new object Array: String[] words = new String[5]; -- this would be an object array of 5 items (0-4)
        - default value of String Array is 'null'.
            - null will throw a null pointer exception.
        - each object in the array must be instantiated separately.

08.04 - 2DArrays

    - an array of arrays
    
    ** 1D arrays store elements (int, String, whatever... simply index of elements)
    ** 2D array is array of arrays, it is more complex than a 1D array

    declared by specifying size of both areas separately:
        int[] [] table = new int[12][50];

    -- arrays stored in one row can be specified by using only 1 index

    -- each array element is then accessed using 2 index values: 
        value = table[3][6];
