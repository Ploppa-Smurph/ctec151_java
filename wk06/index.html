06.04 - Project: Break Dancin'

Create a Java program that uses a for loop and a switch statement to narrate some dance steps. The for loop should iterate from 1 to 10, incrementing by 1 each time. Inside the for loop, create a switch with cases for 5 of the possible numbers created inside the loop as well as a default action. For each of those cases, print out a dance move to the console. For the default case, print out "X claps this time" with X replaced by the number iterated via your for loop. All of your cases should break except for one, so that one dance move in the sequence is followed by another. Some example output: It's dancing time! The Robot! 2 claps right now. The Hustle! 4 claps right now. The Funky Chicken Break it down! 6 claps right now. Break it down! 8 claps right now. And Freeze! 10 claps right now. Requirements Create a for loop that iterates from 1 to 10, including 1 and 10 Create a switch inside that for loop that uses the variable defined in the for loop The switch must contain 5 specific cases matched to different dance moves One of those cases should allow the next defined dance move to activate before breaking The switch should have a default case that prints "X claps right now" with X being replaced by the iterating number The default case is not included in the "5 specific cases" requirement"
