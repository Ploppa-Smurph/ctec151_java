13 - Review

01) A(n) ______________ is an object that serves as a repository for other objects
    Collection

02) A(n) __________________ data type (ADT) is a collection of data and the particular operations that are allowed on that data.
    Abstract

03) A(n)  ______________ list is a dynamic data structure that is implemented using references as links between objects.
    linked

04) A(n) __________ uses first-in, first-out (FIFO) processing.
    queue

05) A(n) _____________ processes information using last-in, first out (LIFO) methodology.
    stack

06) A(n) _________ consists of a root node and optionally many levels of additional nodes that form a hierarchy.
    tree

07) A(n) ___________ consists of key-value pairs.
    map

08) In a _____________ - linked list, a node has a reference to the next node as well as the previous node.
    doubly

09) An abstract data type has a ________, a _________ of values, and a set of operations.
    name, domain
