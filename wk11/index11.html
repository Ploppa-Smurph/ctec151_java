11 - Review

01) A(n) ________________ is an object that defines an unusual or erroneous situation.
    exception

02) A(n) ________________ is different than an exception as it represents an unrecoverable situation and should not be caught.
    error

03) The call _____________ trace indicates where an exception occurred.
    stack

04) A(n) ___________ clause defines how a particular kind of exception is handled and is known as an exception handler.
    catch

05) The ______________ clause is executed no matter how the try block is exited.
    finally

06) True or False: A try block does not need to have a catch clause associated with it.
    True

07) Exception _______________ is when the flow of control returns to the method that called it after an uncaught exception is thrown.
    Propagation

08) The exception _____________ hierarchy define various exceptions related by inheritance.
    class

09) A(n)  ______________ statement is used to begin exception propagation.
    throw

10) A(n) ______________ exception requires no throws clause. An example would be the RuntimeException and any of its descendants in the class hierarchy.
    unchecked
